   - Crank length: label "Crank length", unit mm, guidanceLevel "simple", guidance text: "Check the inside of your crank arm — the length is usually printed there (e.g. 170, 172.5, 175).", placeholder "172.5", helperText "Optional — defaults to 172.5mm (most common road bike crank)" (per D-14).
