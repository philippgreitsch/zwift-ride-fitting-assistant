  <action>
    **File 1: `src/types/fit.ts`** -- Define all TypeScript types per RESEARCH.md Pattern 1 and user decisions.

    Export these exact types:
    - `InputSource = 'fit-report' | 'measured' | 'derived' | 'estimated'` (per D-08)
    - `HandlebarType = 'drop' | 'flat'`
    - `FitInputs` interface with four optional groups:
      - `fitReport?: { saddleHeight?: number; saddleForeAft?: number; handlebarHeight?: number; handlebarReach?: number; }` (all in mm)
      - `physical?: { saddleHeight?: number; saddleForeAft?: number; handlebarHeight?: number; handlebarReach?: number; crankLength?: number; handlebarType?: HandlebarType; dropBarHoodHeightOffset?: number; dropBarHoodReachOffset?: number; }` (all in mm)
      - `body?: { inseam?: number; torso?: number; arm?: number; }` (all in mm)
      - `frame?: { stack?: number; reach?: number; seatTubeAngle?: number; }` (mm and degrees)
    - `AxisOutput` interface per D-10: `{ ideal_mm: number; achievable_mm: number; letter_position: string | null; out_of_range: boolean; direction?: 'above' | 'below'; source: InputSource; confidence: 'high' | 'medium' | 'low'; }`
    - `FitOutputs` interface: `{ saddleHeight: AxisOutput | null; saddleForeAft: AxisOutput | null; handlebarHeight: AxisOutput | null; handlebarReach: AxisOutput | null; allAxesOutOfRange?: boolean; }` (per D-11)

    **File 2: `src/lib/zwiftRideConstants.ts`** -- Encode all hardware constants per RESEARCH.md Pattern 2.

    Export these exact constants with inline source citation comments:
    - `ZWIFT_RIDE_CRANK_MM = 170 as const` // Source: DC Rainmaker review -- HIGH confidence
    - `DEFAULT_CRANK_LENGTH_MM = 172.5 as const` // Per D-14: most common road bike crank
    - `SADDLE_HEIGHT_RANGE = { minMm: 599, maxMm: 865 } as const` // Source: geometrygeeks.bike -- MEDIUM confidence
    - `HANDLEBAR_HEIGHT_RANGE = { minMm: 863, maxMm: 1024 } as const` // Source: geometrygeeks.bike -- MEDIUM confidence
    - `SADDLE_FORE_AFT_RANGE = { minMm: 0, maxMm: 100 } as const` // Source: community forum -- LOW confidence, UNVERIFIED
    - `HANDLEBAR_REACH_RANGE: { minMm: number | null; maxMm: number | null } = { minMm: null, maxMm: null }` // UNVERIFIED -- must be physically measured
    - `ZWIFT_RIDE_SEAT_TUBE_ANGLE_DEGREES = 73.5 as const` // Source: TrainerRoad forum -- LOW confidence
    - `SEATPOST_BUILTIN_SETBACK_MM = 20 as const` // Source: community forum -- MEDIUM confidence
    - `SADDLE_RAIL_SLIDE_MM = 35 as const` // Source: Mountain Massif review -- MEDIUM confidence
    - `DEFAULT_DROP_BAR_HOOD_REACH_OFFSET_MM = 50 as const` // Default estimate, user-overridable
    - `DEFAULT_DROP_BAR_HOOD_HEIGHT_OFFSET_MM = 40 as const` // Default estimate, user-overridable
    - Four empty letter-to-mm lookup tables (per D-03, Pitfall 1):
      - `SADDLE_HEIGHT_LETTER_TO_MM: Record<string, number | null> = {}` with comment: `// UNVERIFIED -- measure on physical hardware. DO NOT derive from range arithmetic.`
      - `HANDLEBAR_HEIGHT_LETTER_TO_MM: Record<string, number | null> = {}`
      - `HANDLEBAR_REACH_LETTER_TO_MM: Record<string, number | null> = {}`
      - `SADDLE_FORE_AFT_LETTER_TO_MM: Record<string, number | null> = {}`

    **File 3: `src/lib/validators.ts`** -- Create Zod v4 schemas for input validation.

    Import `z` from `zod`. Create and export these schemas:
    - `fitReportSchema`: z.object with optional number fields for saddleHeight, saddleForeAft, handlebarHeight, handlebarReach. Each field is `z.number().positive().optional()`.
    - `physicalMeasurementsSchema`: z.object with:
      - `saddleHeight`: `z.number().min(400).max(1000).optional()` (reasonable saddle height range in mm)
      - `saddleForeAft`: `z.number().min(0).max(200).optional()`
      - `handlebarHeight`: `z.number().min(600).max(1200).optional()`
      - `handlebarReach`: `z.number().min(200).max(800).optional()`
      - `crankLength`: `z.number().min(140).max(200).optional()` (covers all crank sizes)
      - `handlebarType`: `z.enum(['drop', 'flat']).optional()`
      - `dropBarHoodHeightOffset`: `z.number().min(0).max(100).optional()`
      - `dropBarHoodReachOffset`: `z.number().min(0).max(150).optional()`
    - `bodyMeasurementsSchema`: z.object with:
      - `inseam`: `z.number().min(500).max(1100).optional()` (mm)
      - `torso`: `z.number().min(300).max(800).optional()` (mm)
      - `arm`: `z.number().min(400).max(900).optional()` (mm)
    - `frameGeometrySchema`: z.object with:
      - `stack`: `z.number().min(400).max(700).optional()` (mm)
      - `reach`: `z.number().min(350).max(450).optional()` (mm)
      - `seatTubeAngle`: `z.number().min(70).max(80).optional()` (degrees)
    - `fitInputsSchema`: z.object combining all four sub-schemas as optional fields: `{ fitReport: fitReportSchema.optional(), physical: physicalMeasurementsSchema.optional(), body: bodyMeasurementsSchema.optional(), frame: frameGeometrySchema.optional() }`
